## Features
- Responsive design for mobile, tablet and desktop
- SEO tags (title and meta description) on every page
- Image alt text for all major images
- Sticky navigation bar with mobile-friendly menu
- View switcher to simulate Desktop / Tablet / Mobile layouts
- Registration and contact form validation using JavaScript
- FAQ accordion using JavaScript
- Shared header, footer and styling across all pages

## How to Run
Open `index.html` in any browser.

## Netlify Deployment Steps
1. Create a GitHub repository.
2. Upload all files from this folder (HTML files, `style.css`, `script.js`, `README.md`).
3. Login to Netlify.
4. Click **"Add new site"** > **"Import an existing project"**.
5. Connect your GitHub repository.
6. Deploy the website.
